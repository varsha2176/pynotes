Word-Count-In-String

Sat 21 June 2025

Count the frequency of each word in a sentence.

text = input("Enter a sentence: ").lower()
words = text.split()
word_count = {}

for word in words:
    word_count[word] = word_count.get(word, 0) + 1

print("Word Frequencies:")
for word, count in word_count.items():
    print(f"{word}: {count}")
